# 人脸情绪识别项目完整实现方案

本文章将教你配置实习作业-xxx老师的人脸情绪识别实训作业的环境配置和基础配置。

使用的工具为VS Code，如果你用的是PyCharm，应该也差不多，有问题自行AI，有不懂的名词也自行AI。

> [!NOTE]
> Python版本是3.11版本的，如果不是，也没有怎么办，用uv命令
（没有pip下载一个，pip install -u uv）。
然后更新
pip install uv --upgrade

## 环境准备
uv下载3.11版本python


```bash
uv python install 3.11   
```
配置3.11版本环境

```bash
uv venv --python 3.11 .venv  
```

## 激活Python环境

```bash
.\\.venv\\Scripts\\Activate.ps1 
```

## 验证Python版本

```bash
python -V 
```

是3.11.14就成功了

## 安装基础依赖

```bash
uv pip install -U pip setuptools wheel  
```

## 安装PaddlePaddle
安装飞浆3.2.0显卡版本
```bash
uv pip install paddlepaddle-gpu==3.2.0 --index-url https://www.paddlepaddle.org.cn/packages/stable/cu126/  
```

或者CPU版本
```bash
uv pip install paddlepaddle==3.2.0 --index-url https://www.paddlepaddle.org.cn/packages/stable/cpu/      
```


## 验证GPU安装

```bash
python -c "import paddle; paddle.set_device('gpu'); x=paddle.randn([1024,1024]); y=paddle.matmul(x,x); print('ok', y.place)"   
```

什么，你看不懂结果，复制上面的命令问AI就行了

这环境就配置好了，之后照着PDF来，

## 数据采集

```bash
python data_collector.py  
```

运行这个会调用摄像头，然后会自动创建emotion_dataset文件夹和里面的情绪文件夹

确保目录文件都是英文，否则会报错


## 目录结构

![Pasted image 20251228150325](/images/posts/cloud-computing/人脸情绪识别制作/20251228150325.webp)

```bash
python -c "import data_collector as dc; dc.preprocess_data()"
```

运行该命令生成.pkl训练文件


## 模型训练
开始训练
```bash
python model_train.py 
```

如果上面开始训练报错，可能要更新cudnn
运行该文件更新cudnn

```bash
uv pip install -U "nvidia-cudnn-cu12>=9.9,<10" -i https://pypi.tuna.tsinghua.edu.cn/simple
```
