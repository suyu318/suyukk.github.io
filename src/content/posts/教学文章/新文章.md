# 如何上传文章到 Mizuki 博客

这篇文章教你如何写新文章并上传到你的 Mizuki 博客。

## 快速开始（三步法）

### 第一步：创建文章文件

在 `src/content/posts/` 目录下创建一个新的 Markdown 文件，比如 `我的第一篇文章.md`。

### 第二步：编写文章内容

在文件中填入文章的 Front-matter（元数据）和正文内容（下面会详细说明）。

### 第三步：推送到 GitHub

```bash
git add .
git commit -m "新增文章：我的第一篇文章"
git push
